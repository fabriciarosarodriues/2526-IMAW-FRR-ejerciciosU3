Ejercicios de PHP

Ejercicio 1.1

Descripción:

  • Crea una variable donde almacenes tu nombre.
  • Muestra la siguiente información:

1-Tipos de variables
Hola Raúl, Bienvenido

Para mostrar el contenido de la tercera línea, debes concatenar el texto “Hola “, la variable donde has almacenado tu nombre y el texto “, Bienvenido”.

Ejercicio 1.2

Descripción:

  • Crea dos variables de nombre a y b que contengan números.
  • Muestra el resultado de las operaciones suma, resta, multiplicación, división y resto.

Resultado esperado cuando a=8 y b=3:

a = 8
b = 3
a + b = 11
a - b = 5
a * b = 24
a / b = 2.6666666666667
El resto de dividir a entre b es 2
Ejercicio 1.3

Descripción:

  • Crea una variable de tipo array con los 12 meses del año y llámale meses.
  • Crea otra variable mes donde almacenes un número del 1 al 12.
  • Muestra la información: El mes 4 es Abril.
Ejercicio 1.4 (opcional)

Descripción:

  • Crea un array coches con las marcas: Renault, Seat, Audi y Mercedes.
  • Crea una variable bola y asigna un valor aleatorio entre 0 y 3.
  • Muestra el coche seleccionado aleatoriamente usando la combinación de coches y bola.

Ejemplo: El coche que te ha tocado es un Audi, Felicidades!!!!
Cada vez que recargues la página, la marca del coche puede variar.
